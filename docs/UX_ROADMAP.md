# CairnGraph UX Roadmap

CairnGraph v1.0 established live-provider verification. The next phase turns CairnGraph into a graph-native explorer.

## v1.1-alpha.1

Status: green, Actions verified.

Scope:

```text
embedded SVG graph explorer
node selection sync
edge filtering
selected-neighborhood mode
standalone renderer compatibility
```

## v1.1-alpha.2

Status: green, Actions verified.

Scope:

```text
zoom controls (zoom-in, zoom-out, reset-view)
pan controls (drag background to pan)
dynamic viewBox for viewport management
```

## v1.1-alpha.3

Status: implemented, pending CI verification.

Scope:

```text
blast-radius visual overlay
affected node highlighting (root, impacted)
blast edge visualization
dim unaffected nodes control
blast summary panel
preserved UX (zoom, pan, filters, selection)
```

## v1.1-alpha.4

Scope:

```text
source and ref inspector (GitHub links)
stone detail drawer
copy evidence payloads
blast history/replay
keyboard shortcuts
```

## v1.1-beta

Scope:

```text
live CairnStone chain explorer polish
multi-chain navigation plan
release candidate UX docs
```
